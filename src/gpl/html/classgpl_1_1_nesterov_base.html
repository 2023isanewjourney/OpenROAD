initiates a nesterovbase object with pb_ (PlacerBase) The information initiated includes:

  • std/macro instance areas, location
  • gPins
  • gCells
  • gNets
  • virtual filler GCell In addition, it initializes bins and bin grid structure Lastly, it initializes fft structrue based on bins
