◆ doInitialPlace()

void gpl::Replace::doInitialPlace ( )

runs a Bi-Conjugate gradient stabilized placement on CPU or GPU which is configurable.

It creates an InitialPlace object. Then, to process the object, under the hood, it runs a series of functions:

  • set center location for instances;
  • set Ext Id and reset Pin Min Max attributes
  • finally, it creates Sparse Metrixs from a PlacerBaseCommon object, which supposes to include everything from PlacerBase that is not region specific. Then, BiCGSTAB Eigen Solver is used to compute the Eigen Vectors for X and Y dimensions.
